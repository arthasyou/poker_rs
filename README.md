# poker_rs

Poker_RS is a rust library.

## poker

contains base of poker

### Installation

1. Clone the repository:
    ```sh
    git clone https://github.com/arthasyou/poker_rs.git
    cd poker_rs
    ```

2. Build the project:
    ```sh
    cargo build
    ```


3. Run the project:
     ```sh
    cargo run
    ```
